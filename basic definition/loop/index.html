Vòng lặp

  • Vòng lặp được sử dụng để thực hiện các thao tác được lặp lại nhiều lần
  • Các vòng lặp hay sử dụng trong lập trình thi đấu
    • For
      • for (int i = 0; i < n; i++) {
    // Thao tác
}
    • While
      • while (condition) {
    // Thao tác
}
    • Do While
      • do {
    // Thao tác
} while (condition);
    • Vòng lặp lồng nhau
      • for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        // Thao tác
    }
}
    • Vòng lặp vô hạn
      • while (true) {
    // Thao tác
}
    • Vòng lặp với điều kiện dừng
      • for (int i = 0; i < n; i++) {
    if (condition) {
        break; // Dừng vòng lặp
    }
    // Thao tác
}
    • Vòng lặp với điều kiện tiếp tục
      • for (int i = 0; i < n; i++) {
    if (condition) {
        continue; // Bỏ qua phần còn lại của vòng lặp
    }
    // Thao tác
}
