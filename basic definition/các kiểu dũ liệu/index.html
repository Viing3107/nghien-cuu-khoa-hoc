Câu lệnh điều kiện

Câu lệnh điều kiện viết bằng ngôn ngữ lập tình C++:

#include <iostream>
using namespace std;
int main() {
    int a;
    cin >> a;
    if (a > 0) {
        cout << "a là số dương" << endl;
    } else if (a < 0) {
        cout << "a là số âm" << endl;
    } else {
        cout << "a là số không" << endl;
    }
    return 0;
}
